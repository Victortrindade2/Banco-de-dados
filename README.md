# 📚 Aula de Banco de Dados

Este repositório foi criado para apoiar os estudos da disciplina de **Banco de Dados**, reunindo materiais, exemplos práticos e exercícios para facilitar o aprendizado dos conceitos fundamentais.

---

## 🧠 Conteúdo Abordado

* Introdução a Banco de Dados
* Modelagem de Dados (Modelo Entidade-Relacionamento - MER)
* Modelo Relacional
* Linguagem SQL

  * DDL (Data Definition Language)
  * DML (Data Manipulation Language)
  * DQL (Data Query Language)
* Normalização de Dados
* Chaves (Primária, Estrangeira, etc.)
* Índices e Performance
* Transações e Controle de Concorrência

---

## 🛠️ Tecnologias Utilizadas

* SGBDs (Sistemas de Gerenciamento de Banco de Dados), como:

  * MySQL
  * PostgreSQL
  * SQLite
* Ferramentas de modelagem (ex: MySQL Workbench, BR Modelo)

---

## 📌 Pré-requisitos

* Conhecimentos básicos de lógica de programação
* Ter um SGBD instalado (MySQL, PostgreSQL ou outro)

---

O objetivo deste repositório é servir como material de apoio para:
